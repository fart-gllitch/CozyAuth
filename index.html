How Cozy Works

Get started with just a few lines of code. It's that simple.

1

Download the Cozy DLL

Get our lightweight DLL and add it to your project. Supports C++, C#, Python, and more.

2

Initialize with Your API Key

Create an account, get your API key, and initialize Cozy in your application.

// C# Example
using Cozy.Auth;

var auth = new CozyClient("YOUR_API_KEY");
bool isValid = await auth.ValidateLicense(license, username);
3

Let Cozy Handle the Rest

Our DLL handles all API calls, encryption, and authentication flows automatically. No more authentication headaches.
